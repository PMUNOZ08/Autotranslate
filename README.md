[![Swift Version][swift-image]][swift-url]

Sample code to create strings catalog programmatically.

This repository is part of [NSCoder Night Madrid](https://nscoder-mad.tumblr.com) Talk "Translation".

The repository include:

- Talk Slides.
- Initial Project 
- Final Project 
- Snippets used in initial project

The sample project is an app that allows translate a list of strings to the selected languages and generate a strings catalog to use it directly in app.
